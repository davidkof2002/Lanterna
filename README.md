# 💡 Keyboard Light — Lanterna para Teclado

## 📌 Sobre o projeto

O **Keyboard Light** é um aplicativo desktop desenvolvido em Python com o objetivo de resolver um problema real: a dificuldade de visualizar o teclado em ambientes com pouca iluminação.

A ideia surgiu a partir de uma necessidade pessoal, quando precisei digitar no escuro em um notebook sem teclado iluminado. Ao invés de buscar uma solução externa, decidi criar meu próprio aplicativo.

O resultado foi uma ferramenta simples, leve e funcional, que atua como uma **lanterna digital para o teclado**, permitindo ajustar a iluminação diretamente na tela.

---

## 🎯 Objetivo

Criar uma solução prática e acessível para melhorar a visibilidade do teclado em ambientes com pouca luz, utilizando programação e conceitos de interface gráfica.

Este projeto também demonstra:

- Resolução de problemas do dia a dia
- Desenvolvimento de aplicações desktop
- Manipulação de interface gráfica (GUI)
- Controle de eventos com mouse e teclado
- Criação de software útil e funcional

---

## 🚀 Funcionalidades

- Barra de luz ajustável (modo slim profissional)
- Controle de intensidade da iluminação
- Alternância entre luz branca e luz quente
- Janela flutuante
- Redimensionamento com o mouse
- Posicionamento fixo e configurável
- Botão para manter sempre na frente
- Atalho de teclado (F9)
- Fechamento rápido com ESC
- Interface leve e responsiva
- Pontas arredondadas para melhor visual

---

## 🖥️ Tecnologias utilizadas

- Python
- PyQt5
- Programação orientada a eventos
- Interface gráfica (GUI)
- Desenvolvimento de aplicações desktop
